## Key Features

- Easy development of typesafe RPC workers and Durable Objects
- Typesafe WebSockets client
- Relatively lightweight and very few dependencies (cookies, devalue, mrmime)
- Works with Zod, ArkType, Valibot out of the box
- Very little boilerplate
- Familiar syntax to create endpoints
- Handle file transfer and many common data types like Map and Set Blob etc.
- Typesafe stream responses
- Great error handling
- Very hookable
- Uses websockets hibernation to reduce costs
- Batteries included (cors, cookies, static files)
- Add and access all your Cloudflare bindings
